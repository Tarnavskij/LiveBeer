# LiveBeer — Тестовое задание (Jetpack Compose)

Тестовое задание: сверстать несколько экранов мобильного приложения по предоставленным макетам.  
Все экраны реализованы на **Jetpack Compose** с акцентом на точное соответствие дизайну: типографика, отступы, цвета, поведение элементов и кастомные компоненты.

---

## 📱 Реализованные экраны

### Welcome Screen
Экран приветствия с большим изображением и тремя кнопками: Вход, Регистрация, Войти без регистрации.

### Login Screen
Экран ввода номера телефона.

- Форматирование номера в реальном времени: `+7 (XXX) XXX XX XX`
- Кастомная цифровая клавиатура в стиле iOS
- Задержка с имитацией запроса, захардкоженная проверка номера
- Отображение ошибок, ссылка на регистрацию

### Register Screen
Экран регистрации нового пользователя.

- Поля: номер телефона, имя, дата рождения (через `DatePickerDialog`), чекбокс согласия
- Валидация формы: кнопка активна только при заполненных полях и принятом согласии
- После успешной регистрации — переход на экран ввода кода

### Code Screen
Экран ввода 4‑значного кода активации.

- 4 кастомные ячейки ввода (точка / цифра / ошибка)
- Таймер обратного отсчёта 47 секунд, повторная отправка кода
- Кастомная цифровая клавиатура
- Задержка с имитацией проверки, захардкоженный код `1111`

### Main Screen (Unauthorized)
Экран для неавторизованного пользователя с белым контентным блоком, фоновыми иллюстрациями, кнопкой «Войти» и кастомным нижним навбаром.

---

## 🧭 Навигация

Навигация реализована через `NavHost`. Экран `code` используется как общий — на него попадают и после логина, и после регистрации.

Маршруты: `welcome` → `login` / `register` → `code` → `main_unauth`

---

## 🎨 Тема

Кастомная тема `LiveBeerTheme` на базе Material 3. В `MainActivity` настроены прозрачный статус-бар и тёмный navigation bar.
