# LiveBeer — Тестовое задание (Jetpack Compose)

Это тестовое задание, в рамках которого необходимо было сверстать 2–3 экрана мобильного приложения по предоставленным макетам.  
Все экраны реализованы на **Jetpack Compose**, с точным соблюдением:

- типографики
- отступов
- цветовой схемы
- поведения элементов
- кастомных UI‑компонентов (клавиатуры, ячеек кода, навбара)

Проект построен модульно, с чистыми Composable-функциями и аккуратной навигацией.

---

## 📱 Реализованные экраны

### 1. Welcome Screen
Экран приветствия с большим изображением, заголовком и тремя кнопками:

- Вход
- Регистрация
- Войти без регистрации

Особенности:

- полноэкранная иллюстрация
- точное соответствие макету
- кастомные цвета и типографика
- навигация через callback-и

---

### 2. Login Screen
Экран ввода номера телефона.

Функциональность:

- форматирование номера в реальном времени: `+7 (913) 210 95 82`
- кастомная цифровая клавиатура в стиле iOS
- валидация номера
- имитация запроса на сервер
- отображение ошибок
- ссылка на регистрацию

Особенности:

- клавиатура полностью кастомная
- кнопка «Далее» активируется только при 10 цифрах
- плавная логика обработки ошибок

---

### 3. Register Screen
Экран регистрации нового пользователя.

Поля:

- номер телефона
- имя
- дата рождения (через DatePickerDialog)
- чекбокс согласия

Функциональность:

- валидация всех полей
- отображение ошибок
- форматирование даты
- кнопка активируется только при валидной форме

Особенности:

- часть текста оформлена как кликабельная ссылка
- кастомные цвета, скругления, отступы

---

### 4. Code Screen
Экран ввода 4‑значного кода активации.

Функциональность:

- 4 кастомные ячейки ввода
- отображение точки, цифры или ошибки
- таймер обратного отсчёта (47 секунд)
- повторная отправка кода
- кастомная цифровая клавиатура
- имитация проверки кода

Особенности:

- нижняя линия вместо рамки (как в макете)
- подсветка ошибок
- кнопка активируется только при 4 цифрах

---

### 5. Main Screen (Unauthorized)
Экран, который видит гость.

Содержит:

- белый контентный блок с фоновыми иллюстрациями
- заголовок и подзаголовок
- кнопку «Войти»
- кастомный нижний навбар

Особенности:

- навбар полностью кастомный
- активный пункт подсвечивается жёлтым
- иллюстрации накладываются слоями

---

## 🧭 Навигация

Навигация реализована через `NavHost`.

Маршруты:

- welcome
- login
- register
- code
- main_unauth

Каждый экран получает только необходимые callback-и.

---

## 🎨 Тема и стили

Используется кастомная тема `LiveBeerTheme`.

В `MainActivity`:

- прозрачный статус-бар
- тёмный navigation bar
- корневой Surface с fillMaxSize()
